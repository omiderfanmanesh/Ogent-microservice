Quick Start

Create your first agent with just a few commands:

# Create an agent
curl -X POST "http://localhost:8000/api/v1/agents" \
  -H "Authorization: Bearer your-token" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Assistant",
    "agent_type": "conversational",
    "description": "A helpful assistant",
    "config": {
      "model_name": "gpt-4o",
      "temperature": 0.7
    }
  }'

# Execute the agent
curl -X POST "http://localhost:8000/api/v1/agents/{agent_id}/execute" \
  -H "Authorization: Bearer your-token" \
  -H "Content-Type: application/json" \
  -d '{
    "input": "What is the capital of France?",
    "stream": false
  }'

Or try using our command-line interface:

# List all agents
python -m app.commands.cli list

# Execute an agent
python -m app.commands.cli execute \
  --agent-id {agent_id} \
  --input "What is the capital of France?" \
  --stream

WebSocket Streaming

For real-time updates during agent execution, connect to:

ws://localhost:8000/api/v1/executions/{agent_id}/stream?token=your-token

This will provide real-time updates as your agent executes, including status changes, steps, command execution, and generated text.
